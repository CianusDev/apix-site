# Installation

apix is distributed as a single self-contained binary. Pick the method that works best for you.

## One-liner

The fastest way to install apix on **Linux** and **macOS**:

```bash
curl -fsSL https://raw.githubusercontent.com/CianusDev/apix/main/install.sh | bash
```

The script auto-detects your platform (x86_64 / ARM64) and installs the binary to `~/.local/bin`.
Make sure `~/.local/bin` is in your `$PATH`:

```bash
echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

Then verify the installation:

```bash
apix --version
```

## Manual download

Grab the latest binary for your platform from the [Releases page](https://github.com/CianusDev/apix/releases):

| Platform | File |
|----------|------|
| Linux x86_64 | `apix-vX.Y.Z-x86_64-unknown-linux-gnu.tar.gz` |
| macOS Intel | `apix-vX.Y.Z-x86_64-apple-darwin.tar.gz` |
| macOS Apple Silicon | `apix-vX.Y.Z-aarch64-apple-darwin.tar.gz` |
| Windows x86_64 | `apix-vX.Y.Z-x86_64-pc-windows-msvc.zip` |

Extract and install on Linux / macOS:

```bash
tar xzf apix-*.tar.gz
mv apix ~/.local/bin/
chmod +x ~/.local/bin/apix
```

On Windows, extract the `.zip` and move `apix.exe` to a directory that is in your `PATH`.

> **Note:** On Windows, a UNIX-compatible terminal such as WSL or Git Bash is recommended for the best experience.

## Build from source

Requires [Rust](https://rustup.rs) (edition 2024):

```bash
git clone https://github.com/CianusDev/apix.git
cd apix
cargo build --release
./target/release/apix
```

You can then copy the binary to your PATH:

```bash
cp target/release/apix ~/.local/bin/
```
